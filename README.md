# Ayu-Heal

Ayu-Heal is a modern interactive web application based on the principles of Ayurveda. It functions as a digital encyclopedia of medicinal plants, providing detailed information about their uses, benefits, preparation methods, dosages, and safety guidelines.

The application includes an integrated assistant called AyurBot, which helps users retrieve plant-specific information in a conversational format.

---

## Overview

Ayu-Heal is designed as a lightweight, framework-free web application built entirely with standard web technologies. It focuses on accessibility, usability, and structured presentation of Ayurvedic knowledge.

The platform allows users to explore medicinal plants, search by disease or plant name, filter by body systems, and save favorite herbs for quick access.

---

## Features

### 1. Comprehensive Herb Database
- Covers 50 Ayurvedic medicinal plants.
- Includes:
  - Common name
  - Latin name
  - Parts used
  - Treated conditions
  - Preparation methods
  - Dosage information
  - Safety guidelines and contraindications

### 2. AyurBot (Intelligent Assistant)
- Built-in chatbot integrated into the application.
- Responds to user queries such as:
  - “Tell me about Ashwagandha”
  - “What helps with diabetes?”
- Fetches relevant data directly from the internal herb database.
- Provides structured, conversational responses.

### 3. Search and Filtering
- Search by:
  - Exact plant name
  - Latin name
  - Treated disease
- Filter herbs by body systems (e.g., Digestive, Respiratory, Nervous).
- Voice search support using `webkitSpeechRecognition`.

### 4. User Experience Features
- Dark and Light mode toggle.
- Smooth theme transition animations.
- Favorites system using browser `localStorage`.
- Persistent storage of:
  - Theme preference
  - Saved favorite herbs
